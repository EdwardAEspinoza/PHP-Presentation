## How to run Ayleen's program:
**1. On Codespace:**
   - Go to ``Ayleen-Trivia-Game`` folder on the terminal
   - Type ``php -S 0.0.0.0:8000``
   - Go to *PORTS* and open 8000 *in browser*

**2. On Local Machine (Windows):**
   - Open *Command Prompt*
   - Go to ``Ayleen-Trivia-Game`` folder location
   - Run ``php -S localhost:8000``
   - Open ``http://localhost:8000/index.php`` on Chrome or similar
